## スコア判定ロジックの詳細調査結果

このプロジェクトのスコア判定は、**ユーザーの描いた線と元画像（お手本）の線とのy座標の違い**を計算しています。以下が詳しい説明です。

### 1. スコア計算の全体フロー

スコア計算は`calculateScore()`関数（game.js, 436-467行）で実行されます：

```javascript
function calculateScore() {
  // 正解の輪郭データ（refTopY, refBottomY）がない場合はスコア0
  if (!refTopY || !refBottomY) return 0;

  const W = CFG.canvas.width;
  const H = CFG.canvas.height;
  const CAT = CFG.cat;
  const SC = CFG.scoring;
  let totalError = 0;
  let count = 0;

  const startPx = Math.floor(CAT.noseX * W);
  const endPx = Math.ceil(CAT.tailX * W);

  // x軸方向に2pxごとのサンプル点で比較
  let idx = 0;
  for (let px = startPx; px <= endPx; px += SC.sampleStep) {
    const xr = px / W;
    const pTop = getTopY(xr, player, W, H);  // プレイヤーの上の線
    const pBot = getBottomY(xr, player, W, H);  // プレイヤーの下の線

    const tTop = refTopY[idx];  // 正解の上の線
    const tBot = refBottomY[idx];  // 正解の下の線
    idx++;

    // y座標の差を二乗して累積
    totalError += (tTop - pTop) ** 2;
    totalError += (tBot - pBot) ** 2;
    count += 2;  // 上下2本の線なので+2
  }

  // RMSEを計算してスコアに変換
  const rmse = Math.sqrt(totalError / count);
  const score = Math.max(0, Math.round(SC.maxScore * Math.max(0, 1 - rmse / SC.maxRmse)));
  return score;
}
```

### 2. 距離計算方法

- **計算対象**: x軸方向に一定間隔でサンプルした点における**y座標のみの差**
- **サンプル間隔**: `SC.sampleStep = 2`（設定値）→ 2ピクセルごとに計算
- **距離指標**: **二乗誤差（差の二乗）の平均→RMSE（平均二乗誤差の平方根）**
  - 計算式: `RMSE = √(Σ(実値 - 計算値)² / サンプル数)`
- **スコア変換式**:
  ```
  score = max(0, round(100 × max(0, 1 - RMSE / 80)))
  ```
  - RMSE = 0 → スコア 100点（完全一致）
  - RMSE ≥ 80 → スコア 0点
  - 設定値: `maxRmse = 80`, `maxScore = 100`

## 原因
