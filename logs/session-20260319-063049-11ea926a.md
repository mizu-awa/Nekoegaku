### 4. スコアリングと評価の仕組み

#### スコア計算（`calculateScore()`：行441-472）

**プロセス：**

1. **サンプル点の比較**
   - x座標：`noseX` から `tailX` まで、2px刻み
   - 各サンプルで正解と プレイヤーのy座標を取得
   - 上下2本の線 → **count = サンプル数 × 2**

2. **二乗誤差の累積**
   ```javascript
   totalError += (tTop - pTop)² + (tBot - pBot)²
   ```

3. **RMSE計算**
   ```javascript
   rmse = √(totalError / count)
   ```

4. **スコア計算**
   ```javascript
   score = max(0, round(100 × max(0, 1 - rmse / 80)))
   ```
   - RMSE = 0 → 100点
   - RMSE ≥ 80 → 0点
   - 線形スケール：rmse 80で0点に到達

#### パラメータの数学的意味

**7つの調整可能なパラメータ：**

| # | パラメータ | 計算式での役割 | 効果 |
|----|-----------|-----------|------|
| 1 | earHeight | ガウシアン振幅 A | 耳の高さ |
| 2 | earGap | 2つのガウシアンの中心間距離 | 耳の間隔 |
| 3 | earWidth | ガウシアン標準偏差 σ | 耳の鋭さ（σ小→鋭い） |
| 4 | tailHeight | しっぽ二次関数の係数 | しっぽの跳ね上げ量 |
| 5 | feetAmp | sin波振幅 | 足の長さ |
| 6 | feetFreq | sin波周波数 ω | 足の本数（周波数大→本数多） |
| 7 | feetPhase | sin波位相 φ | 足の左右位置 |

**各パラメータの計算式：**

```javascript
// 耳の寄与度
ear1 = earHeight × exp(-(x - ear1X)² / (2 × earWidth²))
ear2 = earHeight × exp(-(x - ear2X)² / (2 × earWidth²))

// しっぽの寄与度
tailY = tailHeight × ((t - 0.92) / 0.08)²  (t > 0.92のみ)

// 足の寄与度
feet = feetAmp × sin(feetFreq × 2π × t + feetPhase)
       × envelope(t)  // 顔・お尻でフェードアウト
```

## 問題点の洗い出し

### 1. しっぽが直角に折れている（最大の問題）
お手本のしっぽは滑らかに右上へ跳ね上がるカーブですが、赤線では：

- しっぽの二次関数（`getTailContribution`）が上方向にしか寄与しないため、しっぽの右端が上に飛びすぎている

### 2. お尻の垂直線がお手本と不一致

- **振幅が大きすぎる**: 赤線の足のギザギザがお手本よりも外側にはみ出している
- **波がお手本の内側にも食い込んでいる**: sin波は上下対称に振れるが、お手本の足は**下方向にだけ突出**する形状。sin波の山の部分がベースラインより上に来てしまい、お手本の輪郭内に入り込んでいる
