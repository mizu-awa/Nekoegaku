# ねこえがく — ゲーム設計仕様書

## コンセプト

「数式でねこを描く」パズルゲーム。
画面背景にうっすり表示された「正解のねこ」に合うよう、プレイヤーが数式のパラメータ（スライダー）を制限時間内に調整し、最終的な一致度がスコアとして表示される。

## ねこの構造

ねこは **2本の輪郭線** で構成される（C案）。

### 上の輪郭線（頭〜背中〜しっぽ）

左端（顔の先端）から右端（しっぽの先端）まで1本の連続曲線。

```
y_top(x) = 背中ベースライン
           + 顔の輪郭（左端で下に張り出す二次減衰）
           - 耳ピーク1（ガウシアン）
           - 耳ピーク2（ガウシアン）
           - しっぽ（右端で上に跳ねる二次関数）
```

- **顔の輪郭**: `faceT = max(0, 1 - t*20)` の二乗で左端のみ下方向へ張り出し（耳には影響しない）
- **耳**: `A * exp(-(x - μ)² / (2σ²))` のガウシアン × 2つ
- **しっぽ**: `tailHeight * ((t - 0.92) / 0.08)²` で右端のみ上方向へ

### 下の輪郭線（あご〜足〜しっぽ付け根）

範囲: 顔の先端（noseX = 0.025）からしっぽの先端（tailX = 0.96）まで。

```
y_bottom(x) = ベースライン + 足エンベロープ × sin波
```

- **ベースライン**: 左端は顔先端Y(0.445)からsmoothstepで足の底(bottomY=0.78)へ遷移、中央は水平(bottomY固定)、右端（t>0.96）で急上昇してしっぽ先端Yへ合流
- **足のsin波**: `A * sin(ω * 2π * t + φ)`（顔・お尻領域ではエンベロープでフェードアウト）

### 接続

- 左端: 上の線と下の線が同じ点（顔の先端、noseY = 0.445）で合流
- 右端: 下の線が急上昇してしっぽ先端付近で上の線に合流（お尻の形状）

### 目

耳の下あたりに縦長の楕円を2つ描画（固定位置、パラメータ調整の対象外）。

## プレイヤーが調整するパラメータ（7つ）

| # | パラメータ名 | 数式上の役割 | スライダー範囲 |
|---|-------------|-------------|---------------|
| 1 | 耳の高さ | ガウシアンの振幅 A | 10 〜 150 |
| 2 | 耳の間隔 | 2つのガウシアンの中心間距離 | 10 〜 80 |
| 3 | 耳のとがり | ガウシアンの標準偏差 σ | 8 〜 60 |
| 4 | しっぽの高さ | しっぽの跳ね上げ量 | 0 〜 150 |
| 5 | 足の長さ | sin波の振幅 A | 5 〜 60 |
| 6 | 足の本数 | sin波の周波数 ω | 1.0 〜 14.0 |
| 7 | 足の位置 | sin波の位相 φ | -π 〜 π |

```
[タイトル画面]
  - ゲーム名「ねこえがく」
  - 正解のねこをオレンジ色で表示
  - 「はじめる」ボタン
      ↓
[ゲーム画面]
  - 背景: うっすりオレンジ色の正解ねこ
  - 前面: 黒い線のプレイヤーねこ（リアルタイム更新）

  - スコア（0〜100点）
  - コメント（スコアに応じた5段階）
  - 正解とプレイヤーの重ね描き
  - 「もういちど」ボタン → タイトル画面へ
```

## スコア計算

1. 上の線・下の線それぞれについて、x軸方向に2pxごとのサンプル点でy座標を比較
2. 正解とプレイヤーの差の **RMSE（平均二乗誤差の平方根）** を計算
3. `score = max(0, round(100 * max(0, 1 - rmse / 80)))`
   - RMSE = 0 → 100点（完全一致）
   - RMSE ≥ 80 → 0点

## ステージ構成

現時点では **1ステージのみ**。正解パラメータは固定値。

## 制限時間

**60秒**（暫定。プレイテスト後に調整予定）

## 技術仕様

- **構成**: 単一HTMLファイル（HTML + CSS + JavaScript）
- **描画**: Canvas 2D API（800 × 400px）
- **UI**: HTMLスライダー（input[type="range"]）
- **フォント**: Zen Maru Gothic（Google Fonts）
- **外部依存**: なし（Google Fonts のみ）
- **対応環境**: モダンブラウザ（Chrome, Firefox, Edge, Safari）
